# ShieldMark

> Every dollar you earned, yours.

## What Is ShieldMark

ShieldMark is a transaction integrity system 
built for small businesses. It detects employee 
skimming fraud by comparing what a customer 
actually paid against what the employee logged 
in the system. When a discrepancy is found the 
business owner gets an immediate alert.

## The Problem

Small business owners wear every hat. Manager. 
Marketer. Accountant. They do not have time to 
audit every transaction. That gap is exactly 
where employee skimming hides.

Customer pays $500. Employee logs $350. Pockets 
$150. Repeats daily. Nobody notices.

$150 a day is $54,750 a year. Gone silently.

## How It Works

ShieldMark runs on three layers:

Guard functions validate every transaction input 
before anything else runs.

Helper functions calculate the exact discrepancy 
between actual and logged amounts.

The coordinator orchestrates the full pipeline 
and returns the result to the owner.
